
 Usage

1. Navigate to the home page.
2. Upload your health data file (Excel or CSV).
3. Preview your dataset and select columns for visualization.
4. Generate charts by selecting counties and chart types.
5. Generate forecasts using AI Prophet model.
6. Explore the geospatial map for county-level analysis.
7. Download PDF reports with all metrics for reporting or presentations.
